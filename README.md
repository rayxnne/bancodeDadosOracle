# bancodeDadosOracle
Curso de Banco de Dados Oracle

Este repostório tem como objetivo mostrar conhecimentos adquiridos a no curso de Banco de Dados Oracle da FATEC Santana de Parnaíba.
Foram abordados os seguintes conceitos:
 - Criação e manipulação de tabelas (CREATE, ALTER, DROP).
 - Inserção, atualização e exclusão de registros (INSERT, UPDATE, DELETE).
 - Consultas com SELECT, filtros (WHERE, AND, OR), ordenação (ORDER BY) e eliminação de duplicatas (DISTINCT).
 - Funções de agregação (AVG, SUM, COUNT, MAX, MIN).
 - Relações entre tabelas com JOIN (INNER, NATURAL) e subconsultas.
 - Exercícios práticos envolvendo alunos, cursos, professores, matrículas e ambientes.

A criação do BD e os comandos SQL foram executados utilizando o Oracle Apex.
